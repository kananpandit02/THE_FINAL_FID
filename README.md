# Auction Event – Logistics Checklist

This document lists all logistics, resources, and setup requirements for conducting the Auction Event smoothly.

---

## Human Resources

- Auctioneer 
- UI Operator
- Rule Observer / Judge
- Event Coordinator
- Volunteers (floor coordination)

---

## Technical Setup

- Main laptop (auction UI)
- Backup laptop
- Laptop chargers
- Extension board / power strip
- Projector / LED screen
- HDMI / VGA cable
- Adapter / converter (if required)
- Internet connection (Wi-Fi / hotspot)
- Backup internet source

---

## Software and Data

- Auction UI (fully tested)
- Database access
- Player pool loaded (140 players)
- Team budgets configured
- Undo feature tested
- Backup Excel sheets
  - Participant data
  - Internal tracking

---

## Printed and Physical Materials

- Printed rule book (8 copies)
- Team list (8 teams)
- Prelims result sheet
- Pen and paper
- Timer / stopwatch

---

## Venue Arrangement

- Seating for 8 teams
- Clear projector visibility for all teams
- Auctioneer standing position
- UI operator table near projector

---

## Team Identification

- Team name placards
- Team numbers (1–8)
- Optional badges or stickers

---

## Backup and Safety

- Offline Excel backup
- UI screenshots backup
- Power backup (if available)

---

## Post-Auction

- Playing XI selection setup
- Points calculation verification
- Final leaderboard display
- Winner announcement materials
- Prize or certificate items

---

## Usage

Review this checklist before, during, and after the event to ensure smooth execution.
